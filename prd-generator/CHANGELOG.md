# Changelog

## v1.0.0 - 2026-04-21

### Added
- `prd-generator` skill for generating standardized PRD documents for operations backend pages.
- Support for list pages, form pages, detail pages, and mixed list+form flows.
- Standardized requirement overview table with a one-line core intent summary.
- Fixed writing rules for query items, list sections, operations, forms, detail pages, edit rules, and business flows.
- Built-in common rules for export, column settings, and pagination.
- References and template files for consistent output structure.

### Notes
- Intended for use with concise requirement descriptions and optional prototypes/screenshots.
- Output is optimized for Feishu and DingTalk Markdown rendering.
