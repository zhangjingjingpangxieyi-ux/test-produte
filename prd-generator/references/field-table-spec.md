# 字段表格书写规范

## 1. 表格列定义（7 列）

所有字段表格（查询项、列表、表单）统一使用以下 7 列结构：

| 列名 | 定义 | 填写说明 |
|------|------|----------|
| **字段名** | 页面上展示的字段名称 | 与原型图/设计稿一致 |
| **功能说明** | 这个字段做什么用，简要描述 | 用一句话说明用途，如"输入主播昵称用于搜索" |
| **组件类型** | 前端 UI 组件类型 | 见下方组件类型枚举 |
| **必填** | 是否必填 | 表单填"是/否/条件必填"；查询项和列表填"/" |
| **限制** | 字符数、值范围等硬性限制 | 如"最多50字符""1~999999"；无限制填"/" |
| **数据来源** | 数据从哪来 | 如"手动输入""接口获取""枚举：值1、值2""关联XX表" |
| **备注/说明** | 详细的交互规则 | 按标签规范书写，见第 3 节 |

## 2. 组件类型枚举

| 组件类型 | 说明 | 常用场景 |
|----------|------|----------|
| 输入框 | 单行文本输入 | 名称、标题、ID |
| 搜索框 | 带搜索功能的输入框 | 模糊搜索场景 |
| 数字输入器 | 仅允许数字输入 | 数量、金额 |
| 文本域 | 多行文本输入 | 备注、描述 |
| 富文本编辑器 | 支持格式编排的编辑器 | 邮件内容、公告内容 |
| 下拉框 | 单选下拉 | 类型选择、状态筛选 |
| 下拉框 + 输入框 | 支持下拉选择或输入搜索/新增 | 可扩展的枚举项 |
| 多选下拉 | 多选下拉 | 多标签、多分类 |
| 日历控件 | 日期选择 | 单日期 |
| 日期范围选择器 | 起止日期选择 | 时间范围查询 |
| 日期时间选择器 | 日期 + 时间选择 | 定时发送 |
| 单选 | 单选按钮组 | 二选一、三选一 |
| 多选 | 复选框组 | 多项勾选 |
| 开关 | 开关切换 | 启用/禁用 |
| 上传组件 | 文件/图片上传 | 素材、头像、截图 |
| 物品选择器 | 游戏物品选择 | 邮件附件、奖励配置 |
| 输入框组 | 多个输入框组合 | 设备信息（uuid/IP/归属地） |
| 输入框 + 验证按钮 | 输入后触发验证的组合控件 | 角色ID验证 |
| 文本 | 纯展示文本 | 列表字段展示 |
| 标签 | 带颜色的状态标签 | 状态展示 |
| 按钮组 | 多个操作按钮 | 列表行内操作列 |
| 百分比 | 百分比数值展示 | 比例类数据 |
| 数字 | 数字展示 | 计数类数据 |

## 3. 备注/说明 标签规范

在「备注/说明」列中，按以下标签分类书写，按需选用，不必全部包含：

### 标签定义

| 标签 | 用途 | 书写格式 |
|------|------|----------|
| **【默认值】** | 字段的默认显示/选中值 | 【默认值】{值} |
| **【占位提示】** | 输入框内的灰色引导文案 | 【占位提示】{文案} |
| **【校验规则】** | 校验条件 + 不通过时的提示文案 | 【校验规则】{条件} → 提示："{文案}" |
| **【联动逻辑】** | 与其他字段的联动关系 | 【联动逻辑】{触发条件}时，{联动效果} |
| **【特殊说明】** | 边界情况、业务规则、其他补充 | 【特殊说明】{说明内容} |

### 书写顺序

按 默认值 → 占位提示 → 校验规则 → 联动逻辑 → 特殊说明 的顺序书写。

### 书写示例

**示例 1：简单字段**

> 【占位提示】请输入邮件标题

**示例 2：带校验的字段**

> 【占位提示】请输入主播昵称
> 【校验规则】
> - 最多9个字符，超过部分截断 → 提示："昵称最多支持9个字符"
> - 校验名称唯一性 → 重复时提示："主播昵称已存在，请重新输入"

**示例 3：复杂联动字段**

> 【默认值】关闭
> 【联动逻辑】开启后展开发送时间选择器
> 【特殊说明】关闭时表示立即发送

**示例 4：列表展示字段**

> 【特殊说明】无数据时展示"-"；保留两位小数，如：45.23%

**示例 5：无特殊规则**

> /

## 4. 不同场景的表格差异

| 场景 | 必填列 | 限制列 | 备注/说明列侧重点 |
|------|--------|--------|-----------------|
| **查询项** | 填"/" | 通常填"/" | 【占位提示】【特殊说明】 |
| **列表字段** | 填"/" | 通常填"/" | 【特殊说明】（展示格式、空值处理） |
| **表单字段** | 填"是/否/条件必填" | 填具体限制 | 全部标签都可能用到 |
