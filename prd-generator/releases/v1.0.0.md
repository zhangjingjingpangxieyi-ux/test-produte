# prd-generator v1.0.0 Release Notes

Release date: 2026-04-21

## What this release includes
- Standardized PRD generation for operations backend requirements.
- Supported page types:
  - list pages
  - form pages
  - detail pages
  - mixed list + form pages
- Requirement overview table with:
  - requirement name
  - requirement description
  - requirement type
  - related system/game
  - version
  - priority
  - requester
  - request date
  - one-line core intent summary
- Fixed writing rules for:
  - query items
  - list fields
  - operations
  - forms
  - detail pages
  - secondary edit rules
  - business flow diagrams
- Fixed common rules for:
  - export
  - column settings
  - pagination

## Maintenance notes
- The skill is intentionally opinionated to reduce drift in output structure.
- The template and references are designed to stay aligned with the skill body.
